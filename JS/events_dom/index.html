buscando elementos

metodo descripcion
document.getElementById(id) Find an element by element id
document.getElementsByTagName(name) busca por la etiqueta html, devuelve una lista
document.getElementsByClassName(name) Find elements by class name, retorna una lista
document.querySelectorAll() const x = document.querySelectorAll("p.intro");
retorna una lista de todos los elementos p con class="intro".

cambiando elementos

propiedad descripcion
element.innerHTML = new html content Change the inner HTML of an element
element.attribute = new value Change the attribute value of an HTML element por ejemplo element.id = "id1"
element.style.property = new style Change the style of an HTML element por ejemplo element.style.borderColor = "black";
metodo: element.setAttribute(attribute, value) Change the attribute value of an HTML element

agregando y borrando elementos

Method Description
document.createElement(element) Create an HTML element
document.removeChild(element) Remove an HTML element
document.appendChild(element) Add an HTML element
document.replaceChild(new, old) Replace an HTML element
document.write(text) Write into the HTML output stream
